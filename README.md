🐾 PetMart – Online Pet Store (Django)

PetMart is a full-stack web application built using Django that allows users to browse pets, manage carts and wishlists, place orders, and receive email confirmations. The platform also includes a dedicated admin panel for product and order management. This project demonstrates real-world e-commerce workflows and backend logic using Python and Django.

🚀 Features
👤 User Features

User registration & login authentication

Browse pets by category (Dog, Cat, Bird, Fish, Rabbit)

Search pets by name

Add pets to cart and wishlist

Buy all items in cart with Cash on Delivery

View order summary with order & delivery dates

Email confirmation after successful order

User profile management

🛠 Admin Features

Secure admin login

Add, update, and delete pet products

Manage product stock

View registered users

View and manage customer orders

🧰 Technologies Used

Backend: Python, Django

Frontend: HTML, CSS, Tailwind CSS

Database: SQLite

Authentication: Django Auth

Email Service: SMTP (Gmail)

Version Control: Git
